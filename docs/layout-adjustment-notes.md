# Layout Adjustment Notes

This file records layout corrections requested during visual review so they remain consistent across future sections.

## Hero Alignment Rules

- The `AXTOPANI` wordmark defines the primary left alignment reference for the homepage.
- Section headlines directly below the hero should use the same horizontal inset as the hero wordmark.
- Do not introduce extra container padding that shifts featured section headlines inward more than the hero text.

## Featured Work Rules

- For a single flagship project section, use a strict `50 / 50` desktop split:
  - left half image
  - right half text
- Avoid card-like framing for this spread.
- Keep the section white with black headline text and smaller muted informational text.
- Use thin dividers above the text column or subsection groups.

## Color Rules

- The default site canvas is black.
- Any text that sits directly on the main site background should default to white.
- Only switch text to black when the section itself introduces a white or light background surface.

## Review Memory

- When the user asks to "fix" spacing or alignment, prefer updating these notes and the main style guide so the same issue is not reintroduced in later sections.
- Keep nav alignment, hero alignment, and major section headline alignment visually connected.
